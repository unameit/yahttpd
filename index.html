Welcome to Yet Another HTTP Daemon (yhttpd)

This project was originally created at AGH University of Science and Technology in 1998.

Requirements

  1. HTTP Daemon (a web server) for Linux and any other UNIX flavour (e.g. AIX, Solaris
  2. Support for HTTP protocol version 1.0
  3. Implementation in C++ programming language using functions from POSIX standard
  4. Usage of multiple threads or processes
  5. Usage of interprocess communication
